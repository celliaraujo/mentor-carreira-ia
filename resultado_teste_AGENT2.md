## 🚀 PROJETO DE PORTFÓLIO

**PROJETO:** Aplicação CRUD simples (ex.: cadastro de usuários)

**O QUE FAZER:**  
Criar uma aplicação web que permita cadastrar, listar, editar e excluir usuários.

**ENTREGÁVEIS:**
- Código funcional
- Documentação clara
- Deploy em plataforma gratuita (Heroku, Vercel)

**CRITÉRIOS DE ACEITAÇÃO:**
- CRUD completo funcionando
- Interface simples e intuitiva
- Código versionado no GitHub

**DICA:** Comece pequeno e vá incrementando funcionalidades.
